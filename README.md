Elixir's OneSky API client.

Documentation can be found [here](https://hexdocs.pm/onesky).

## Installation

Add the following to your `mix.exs` file

```elixir
def deps do
  [
    {:onesky, "~> 0.11.0"}
  ]
end
```

And run `mix deps.get`

## Configuration

```elixir
config :onesky,
  api_key: "YOUR_API_KEY",
  api_secret: "YOUR_API_SECRET"
```
